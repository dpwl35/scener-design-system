# SCENE;er Design System

전시·공연·팝업·영화를 발견하고, 나만의 씬(scene)을 기록하고 나누는 문화 콘텐츠 플랫폼 **SCENE;er**의 디자인 시스템 문서입니다.

## 서비스 소개

취향이 있는 사람도, 만들어가고 싶은 사람도 환영하는 공간입니다. 전시·공연·팝업·영화를 장르 구분 없이 발견하고, 직접 후기와 아티클을 발행하며 자신만의 문화적 취향을 만들어갈 수 있습니다.

핵심 기능은 크게 세 가지 축으로 구성됩니다.

- **발견** — 전시·팝업·공연·영화 소식, 에디터 아티클
- **기록** — 개인 아티클 발행, 전시 후기·평점
- **취향** — 내가 쌓아온 씬들이 모여 만들어지는 취향 프로필

## 디자인 원칙

### 1. 다크 베이스, 비비드 포인트

배경은 항상 다크 톤(`#1C1C1C` 계열)을 기본으로 하고, 그 위에 채도 높은 컬러를 포인트로 사용합니다. 어두운 배경일수록 비비드 컬러의 대비가 극대화되어, "패셔너블하고 활기있는" 톤을 가장 효과적으로 전달합니다.

### 2. 색으로 장르를 구분한다

전시·공연·팝업·영화는 항상 같은 색으로 표시됩니다. 유저는 서비스를 쓰다 보면 자연스럽게 "퍼플은 전시, 핑크는 공연"이라는 감각을 익히게 됩니다. 이 규칙은 어떤 화면, 어떤 컴포넌트에서도 깨지지 않아야 합니다.

### 3. 하나의 시그니처 컬러

메인 브랜드 컬러(라임)는 오직 핵심 CTA와 브랜드 아이덴티티(로고, 강조 텍스트)에만 사용합니다. 남용하면 시그니처로서의 힘을 잃습니다.
